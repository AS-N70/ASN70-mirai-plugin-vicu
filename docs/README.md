==
___



### 使用(所有命令前缀都可以不加)
* **#查评论**
  >需要在30秒内输入要查询的用户UID,输入“取消”来取消，或超时自动取消
* **#查评论 用户UID [UP主UID]**
  >[UP主UID]参数可选,输入时查某个用户在指定UP主评论区的评论。否则返回所有评论
